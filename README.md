# Portfolio Website

A modern, responsive personal portfolio website built with HTML, CSS, and JavaScript.

## Files

- `index.html` — page structure and content
- `styles.css` — responsive styling and layout
- `script.js` — mobile navigation and navigation highlighting

## How to use

1. Open `index.html` in a browser.
2. Replace placeholder text with your name, contact details, and project descriptions.
3. Add your own logo or imagery if desired.

## Run with npm

1. Open a terminal in the `Portfolio` folder.
2. Run `npm install` once to install dependencies.
3. Run `npm start`.
4. Open `http://localhost:8000` in your browser.

## Dark mode

- The site defaults to a dark color mode.
- Click the color mode button in the top bar to switch to light mode.

## Profile photo

- The portfolio now defaults to the placeholder image `profile.svg`.
- If you want your actual photo, add it as `profile.jpg` and update the `src` in `index.html` to `profile.jpg`.

## Customize
- Update the hero message in `index.html`.
- Add new sections or project cards.
- Change colors in `styles.css` to match your personal brand.
